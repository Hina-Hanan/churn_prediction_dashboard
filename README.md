# Customer Churn Prediction Dashboard 

**Live Demo:** https://yourname-churn-dashboard.streamlit.app

## What it does
- Predicts churn probability using XGBoost (ROC-AUC 0.84)
- SHAP explanations for every prediction
- Filters + KPIs for business users
- ROI calculations for retention actions

## Tech Stack
- Python | XGBoost | SHAP | Streamlit | Plotly
